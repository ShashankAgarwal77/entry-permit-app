# Jobs to be Done: Gate Security

Format: **When** [situation], **I want to** [motivation], **so I can** [expected outcome].

1. **When** a visitor presents a pass, **I want to** scan it and get an instant Verified / Denied / Expired verdict, **so I can** decide to admit them in seconds.

2. **When** a pass scans as valid, **I want to** see the simulated identity next to the person, **so I can** confirm they are the approved visitor before letting them in.

3. **When** a scan won't go through (dead phone, damaged code), **I want to** have a clear manual fallback, **so I can** still verify the visitor without holding up the line.

4. **When** a pass is expired or already used, **I want to** see that unmistakably, **so I can** refuse entry confidently and consistently.

5. **When** I have to admit someone manually, **I want to** the action to be recorded, **so I can** keep the gate accountable without extra paperwork.
